# Todo App


### Description:  
This app allows users to log in and create todo items.  Once created the items can be deleted, edited, and marked complete.  

### Screenshots
![Image1](todo.png)

### To run:
```
$ cd lib /
$ npm install
$ bower install
$ grunt
```
